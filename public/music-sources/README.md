# 洛雪音乐音源集合

本站提供三个可用的洛雪音乐音源，可直接导入使用。

## 音源列表

1. **LX音源** (`lx.js`)
   - 官方独家音源
   - 支持多平台音乐搜索
   - 稳定更新

2. **六音音源** (`sixyin.js`)
   - 版本：v1.2.1
   - 官网：www.sixyin.com
   - 支持高品质音乐

3. **Huibq音源** (`huibq.js`)
   - 版本：v1.2.0
   - 支持酷我、酷狗、QQ、网易云、咪咕音乐
   - 支持128k/320k音质

## 使用方法

### 本地使用
打开洛雪音乐 → 设置 → 音源设置 → 自定义源 → 导入对应JS文件即可。

### 在线使用
直接填入以下地址即可：
- LX: `https://你的域名/music-sources/lx.js`
- 六音: `https://你的域名/music-sources/sixyin.js`
- Huibq: `https://你的域名/music-sources/huibq.js`
